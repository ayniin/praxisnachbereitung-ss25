CSV-Dateien (Geräte, Ausleihen, Mitarbeiter) in LibreOffice Calc importiert

Verknüpfung der Tabellen über:

PivotTables zur Analyse erstellt (Gerätetyp, Standort, Ausleihanzahl)

Ergebnisse mit Diagrammen visualisiert

Daten anschließend testweise mit pandas in Python eingelesen
